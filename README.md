# Learning Java Concepts

This repository is now set up as a Gradle-based Java project.

## Project Structure

- `src/main/java` for application code
- `src/test/java` for tests
- `build.gradle` for build configuration
- `settings.gradle` for project name

## Quick Start

If Gradle is installed locally:

```bash
gradle clean build
gradle run --args="Java"
```

## Notes

- The project uses Java toolchain version 25 in `build.gradle`.
- Tests are configured with JUnit 5.
